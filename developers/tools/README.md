# Tools

CLI tools for working with Phantasma data and deployments.

## Available tools

- [pha-decode](pha-decode.md) - decode Carbon and VM transactions plus legacy event hex.
- [pha-deploy](pha-deploy.md) - deploy tokens, create NFT series, and mint NFTs.

## When to use these tools

- Use **pha-decode** when you need to inspect raw transactions or legacy event hex.
- Use **pha-deploy** when you want a config-driven alternative to the Token Deployment UI.
